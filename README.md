# 🧪 거상 제작 계산기 (Gersang Calculator)

거상 생산시설 / 제조기술 제작 시  
시장 가격을 기반으로 **제작 비용, 평균가, 이익을 자동 계산**해주는 웹 서비스입니다.

👉 https://gersang-calculator.vercel.app

---

## 🚀 주요 기능

### 📊 제작 계산
- 제작 수량 입력 시 자동 계산
- 작업량 / 게임 시간 / 현실 시간 계산

### 💰 시장 가격 연동
- 실시간 경매장 데이터 기반 가격 반영
- 최저가 + 수량 기반 평균가 계산

### 📈 이익 계산
- 제작 비용 vs 시장 가격 비교
- 이익 / 손해 금액 + 퍼센트 표시

### 🧱 직접 제작 ON/OFF
- 하위 재료를 직접 제작할지 선택 가능

### ⭐ 즐겨찾기
- 자주 사용하는 제작식 저장

### 🔍 검색 기능
- 제작 아이템 / 필요한 재료 아이템 검색 지원

- 필요한 수량만큼 **가중 평균 계산**
